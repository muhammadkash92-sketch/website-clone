# Daraz Website Clone 

Layout & Design

Sticky orange header with logo, search bar, and action buttons
Top info bar (Sell on Daraz, Track Order, Sign In, etc.)
Category navigation rail with active state
3-column hero layout: category sidebar + auto-sliding banner + promo cards
Mini promotional banners row
Category icon grid strip

Core Functionality

 Auto-rotating banner slider with dot indicators and prev/next arrows
 Live countdown timer for Flash Sale section
 Working cart sidebar — add items, change quantity, remove items, live total
 Search bar with enter key support
 Toast notifications on every interaction
Discount badges, star ratings, and sold progress bars on product cards

Responsiveness

Desktop: full 3-column hero, 6-column product grid
Tablet: 2-column layout, sidebar hidden
Mobile: single column, compact nav, full-screen cart panel
